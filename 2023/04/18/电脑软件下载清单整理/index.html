    电脑软件下载清单整理

    本来没打算整理的，结果打算给新电脑腾地方的时候，才发现要下载的软件也太多了。

    考虑到后续可能会有可能继续升级新电脑，所以整理了该下载清单。

    正文

    电脑环境

    最基础的电脑软件，装机必备

    • node，前端开发环境，用官方推荐的版本
    • java8，java开发环境，用8版本比较稳定
    • edge浏览器，微软浏览器
    • chrome，谷歌浏览器
    • Opera浏览器，谷歌内核浏览器
    • bandzip，好用的文件压缩软件

    开发工具

    程序员大多数都会接触到的工具，有些工具的推荐并非我喜欢，而是不得已。

    有时候为了实现需求，往往必须要用一些特殊的工具，没办法。

    • VsCode，开发工具
    • IntelliJ IDEA，后端开发工具
    • hbuilderX，国产多平台代码开发工具
    • gitbash，git命令终端及环境
    • python3，python开发环境
    • ApiPost，接口测试
    • MobaXterm，服务器连接管理
    • PuTTY (64-bit)，服务器连接管理
    • GitHub Desktop，github代码管理工具
    • WinSCP 5.21.2，服务器可视化操作工具

    文档工具

    关于wps，个人觉得，如果有正版平替的话，还是用微软的吧。

    现在微软接入了AI，比起WPS的全家桶，好用了不是一点两点。

    • typora，好用的markdown编辑器（收费）
    • 有道云笔记，工作常用的备忘录
    • wpsoffice，一些文档打开工具
    • 语雀，办公可能会用到

    影音工具

    因为我用的是win系列的，所以这里主要会推荐adobe的工具

    • 网易云音乐
    • 腾讯视频
    • 爱奇艺，
    • PotPlayer，视频播放器
    • 万兴优转，视频格式转码，不推荐
    • 小丸工具箱，视频格式转码
    • Adobe photoshop，图片编辑工具
    • Adobe premiere，视频编辑工具
    • Adobe Effects，视频特效工具

    建模绘图

    • 草图大师（SketchUp），快速建模软件
    • SAI，绘画工具

    游戏工具

    这里大多数都是我自己常用的软件，如果有需要

    • 哔哩哔哩直播姬，bilibili直播软件
    • OBS Studio，电脑直播软件
    • 网易UU，游戏加速器
    • TLY - Baacloud，翻墙梯子（收费）
    • Cheat Engine 7.1，游戏作弊器
    • steam，游戏平台
    • Epic Games，游戏平台
    • wegame，游戏平台
    • origin，游戏平台
    • ubisoft，游戏平台
    • 夜神模拟器，手游模拟器

    其他工具

    腾讯电脑管家和百度网盘这两个东西，说实话，不是很想用。

    但是比起无序扩张的360，有些时候，确实这两个工具还是有必要使用的。

    • 腾讯电脑管家
    • 百度网盘
    • 阿里云盘
    • SynologyDrive，群晖驱动

    • 阿里旺旺

    本清单会不定期更新，毕竟，事物总是不断发展的。
